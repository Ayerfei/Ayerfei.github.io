Valid Parentheses
@ · Oct 12, 2019 · 1 min read · Update at Oct 12, 2019

description：

Given a string containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

An input string is valid if:

  1. Open brackets must be closed by the same type of brackets.
  2. Open brackets must be closed in the correct order.

Note that an empty string is also considered valid.

算法：

一、依次扫描括号序列

  1. 若是左括号，则入栈；
  2. 若是右括号，则检查栈，栈空则右括号多余，栈不为空则匹配结果；

二、 读完所有括号时，检查栈，若栈空则正确匹配。

使用Python进行解题的好处：

  1. list自带栈特性，使用append()和pop()方法实现“后进先出”；
  2. 当s为空时不进入循环，不用提前判断是否为空，节省判断时间。

solution：

def isValid(self, s: str) -> bool:
        key = {'(':')', '[':']', '{':'}'}
        left = ['(','[','{']
        stack = []
        for p in s:
            if p in left:
                stack.append(p)
            else:
                if stack != [] and key[stack[-1]] == p:
                    stack.pop()
                else:
                    return False
        if stack != []:
            return False
        return True
